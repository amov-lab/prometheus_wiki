开始充电
--------

#. 连接充电器与电源
#. 根据相应的电池设置充电器.\ **请确保相应电池使用相应的充电设置,否则可能导致电池报废**

   #. 3s homer电源充电设置,如图
      |arc/1637920515965.jpg|
   #. 6s 无人机电源充电设置,如图
      |arc/1637920515971.jpg|

#. 长按最右边start按键启动(长按时听到铃声则表示一开始充电)
